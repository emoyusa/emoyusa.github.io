>Week 3 Progress

Model

  • "Real" disparities model version 2.1
    • "Calibrated" model to keep populations stable but appreciate that data on prevalence by age provides insight into net changes but not magnitude of inflows and outflows; have set outflows to be constant and set probability of starting to exercise equal to probability of ceasing to be obese
    • Added line to separate east and west, two races, a Disparities Effect variable that gives greater health care benefit to pinks vs. browns, and trackers for different counts and rates
    • Added sliders to allow users to adjust model parameters at setup or as model is run
  • "Educational" model
    • Added background graphic
    • Added 5 levels of increasing difficulty
    • Developed gaming user interface with increasing user actions as levels progress
    • Added ability to keep track of a high score and take snapshot if new high score is attained
